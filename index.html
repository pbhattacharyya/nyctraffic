Discussion

Through this project, we learned that:


  1. Time of day has a big effect on when accidents happen. Unsurprisingly, most crowded hours have the most number of accidents.

  2. Day of the week also have a big effect on the number of accidents. Fridays have the highest number of accidents on average, and Sundays the lowest.

  3. The most extreme data points with highest number of accidents, number of people injured or killed are due to extreme events such as a truck attack on October 31st.

  4. The machine learning models we used told us that high-accident days are really difficult to predict using just weather data we had access to. However, it is possible that bucketing the data differently would yield a better result.

  5. A classifier is likely to be stronger if the variables are not dependent on each other.

Next Steps

Other relationships can be explored using the machine learning models are the effect of certain weather features on specific accident causes. One example could be looking to see if precipitation and snowfall has an effect on the number of accidents caused by “slippery road” or “following too closely.”
